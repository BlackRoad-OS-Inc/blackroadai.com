Capabilities
What we run
Edge-native AI across inference, generation, speech, and vision — all on Raspberry Pi 5 hardware.

AI Gateway

Node.js HTTP gateway (src/index.ts) with rate limiting, CORS, task marketplace, and memory chain. Routes /v1/chat/completions across 8 cluster nodes with auto-failover.

6 Named Agents

Python agent classes: alice.py (worker — execute, generate_code, run_shell), lucidia.py (reasoning — chat, stream, philosophize), octavia.py, prism.py, echo.py, cipher.py.

AI Cluster Gateway

FastAPI gateway (blackroad-ai-api-gateway) defining 8 Pi cluster nodes. /chat with model auto-selection, load balancing, /models, /health, /broadcast endpoints.

CECE Identity

Custom inference models fine-tuned for BlackRoad operations. cece-profile.json identity persistence, modelfiles/ for Ollama custom model definitions.

Persistent Memory

156,675 entries in FTS5 index. 228 SQLite databases. Lucidia's main.py stores key-value memory pairs and dispatches tools across provider registry.

br-ai.sh

Shell tool that connects to Ollama, picks best available model, streams prompts, performs code review and diff summaries. Part of the 80+ br-* CLI tools in the operator monorepo.
